Android-Glossar
DEX (Dalvik Executable)

Binär-Format des Byte-Codes in der Installations-Datei einer Android-App, der von der Dalvik Virtual Machine (DVM) ausgeführt werden kann; neuere Android-Versionen verwenden die neue Laufzeitumgebung Android RunTime (ART), die den Bytecode im DEX-Format einmalig während der Installation in Geräte-spezifischen Maschinencode übersetzt.
